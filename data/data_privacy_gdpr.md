# Data Privacy & GDPR Compliance Policy

## Overview
NexaCloud is committed to protecting the privacy and security of all customer data. This policy explains how we collect, store, process, and protect your data in compliance with GDPR, CCPA, and other applicable privacy regulations.

---

## 1. Data We Collect

### Account Data
- Name, email address, job title
- Billing information (stored tokenized via Stripe — NexaCloud never stores raw card data)
- Company name and address
- IP address and login timestamps (security logging)

### Usage Data
- API request logs (retained 90 days)
- Feature usage analytics (anonymized)
- Error and performance logs (retained 30 days)
- Session recordings (only with explicit consent, optional)

### User-Generated Content
- Files, documents, or data uploaded through our platform
- Support ticket content and attachments
- Chat and message content within our platform

## 4. Your GDPR Rights

If you are in the EU/EEA, you have the following rights:

| Right | Description | How to Exercise |
|-------|-------------|-----------------|
| **Access** | Request a copy of your data | Settings → Privacy → Data Export |
| **Rectification** | Correct inaccurate data | Settings → Profile |
| **Erasure** | Delete your account and data | Settings → Privacy → Delete Account |
| **Portability** | Export data in machine-readable format | Settings → Privacy → Data Export |
| **Objection** | Object to certain processing | privacy@nexacloud.io |
| **Restriction** | Limit processing of your data | privacy@nexacloud.io |

**Response Time:** We respond to privacy requests within **30 days** as required by GDPR.

## 5. Data Retention Policy

| Data Type | Retention Period |
|-----------|------------------|
| Account data | Duration of subscription + 90 days |
| API logs | 90 days |
| Billing records | 7 years (tax compliance) |
| Support tickets | 2 years |
| Deleted account data | 30 days then permanently purged |
| Backup data | 30 days rolling backup |

---

## 6. Data Security Measures

- **Encryption at Rest:** AES-256 for all stored data
- **Encryption in Transit:** TLS 1.3 for all API communications
- **Access Controls:** Role-based access control (RBAC) with principle of least privilege
- **Audit Logging:** All data access logged and retained for 1 year
- **SOC 2 Type II:** Annual third-party security audit
- **Penetration Testing:** Quarterly by certified security firms
- **Data Anonymization:** Analytics data anonymized before processing

## 8. CCPA Compliance (California)

California residents have additional rights under CCPA:
- Right to know what personal information is collected
- Right to delete personal information
- Right to opt-out of sale (NexaCloud does **not** sell data)
- Right to non-discrimination for exercising rights

To exercise CCPA rights: **https://nexacloud.io/ccpa-request**

## 9. Data Breach Notification

In the event of a data breach:
- Affected customers notified within **72 hours** (GDPR requirement)
- Notification includes: nature of breach, data affected, mitigation steps taken
- Regulatory authorities notified as required by applicable law
- Incident reports available at: **https://nexacloud.io/security/incidents**
